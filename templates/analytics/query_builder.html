Query Tips

  • Denomination families: Use "Census Families" for original census groupings (e.g., "Baptist bodies") or "Religious Ecologies Families" for research-based groupings (e.g., "Baptist", "Catholic")
  • Individual denominations: Select specific denominations when you need precise filtering (e.g., "Southern Baptist Convention" instead of all Baptist bodies)
  • Multiple selections: Hold Ctrl (Cmd on Mac) to select multiple items in any dropdown. You can combine filters across tabs.
  • Filter combination: All selected filters are combined (AND logic for different filter types, OR logic within denomination selections)
  • Location: You can filter by state, county, and/or city
  • Export limits: CSV and JSON exports include all matching results (not limited to 100)
  • Admin links: Exported data includes direct links to edit records in Django admin
